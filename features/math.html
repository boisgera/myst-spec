Math & Equations¶

Math Node¶

Specification¶

Math node for presenting numbered equations

  • type*: string (“math”) - See Node

  • identifier: string - See OptionalAssociation

  • label: string - See OptionalAssociation

  • value*: string - See Literal

  • position: object (Position) - See Node

  • data: object - See Node

Example¶

```{math}
:label: matrix
Ax = b
```

type: root
children:
  - type: mystDirective
    kind: math
    options:
      label: matrix
    value: Ax = b
    children:
      - type: math
        identifier: matrix
        label: matrix
        value: Ax = b
(1)¶\[Ax = b\]

Inline Math Node¶

Specification¶

Fragment of math, similar to InlineCode, using role {math}

  • type*: string (“inlineMath”) - See Node

  • value*: string - See Literal

  • position: object (Position) - See Node

  • data: object - See Node

Example¶

This is genius {math}`e=mc^2`

type: root
children:
  - type: paragraph
    children:
      - type: text
        value: 'This is genius '
      - type: mystRole
        kind: math
        value: e=mc^2
        children:
          - type: inlineMath
            value: e=mc^2

This is genius \(e=mc^2\)
